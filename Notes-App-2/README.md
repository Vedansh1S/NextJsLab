# Notes App 2

Server-side notes application using Next.js Server Actions.

## Features

- Add and delete notes
- Server Actions for data management
- Server-side rendering

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000)
