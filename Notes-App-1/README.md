# Notes App 1

Client-side todo application with modern glassmorphism UI.

## Features

- Add and manage todos
- Glassmorphism design
- Client-side state management

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000)
